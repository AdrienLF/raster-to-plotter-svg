Composing the page

Layers are the whole game: every imported image, path-finding result, generator drawing and SVG is a layer on the page. The Composition step is where they become one picture.

The Layers panel

  • Import / ＋ Path finding / ＋ Generator / ＋ Cavalry — raster images, image-driven strokes, procedural drawings, or a live bridge (below).
  • Checkbox — visibility. Hidden layers are excluded from export and plot.
  • ⚙ — reopens the layer's Path Finding settings; every layer remembers its module + parameters and can regenerate any time.
  • ↑ ↓ — stacking order. Later/upper layers draw on top and can occlude below.
  • ⧉ / × — duplicate / delete. The name field renames. Deleted something good? ⌘Z / Edit → Undo brings the whole composition back (also after an unwanted regenerate).
  • Display — show a layer as its source raster, its strokes, or both, while you compose.

Placing layers

Drag a layer on the canvas to move it; edges snap to the page and to other layers' edges and centers (guide lines flash as you hit them). The toolbar's six align buttons align the selected layer to the page: left / center / right, top / middle / bottom. Grab a corner handle to scale, or enter an exact X, Y, width, height, scale, or rotation in the panel. Area → Fit keeps the whole layer inside the padded drawing area; Fill covers it. Pen widths stay true — only the geometry scales.

Importing raster images

File → Import image… (or the 🖼 rail button) creates a new raster layer instead of replacing or cropping the page. PNG, JPEG, BMP, TIFF and WebP imports preserve transparency and camera/phone EXIF orientation. The complete image is fitted and centred initially, but it remains independent: move, scale, rotate, duplicate and stack it like any other layer.

Click the layer's ⚙ to run path finding on its own full, uncropped pixels. Choose Raster, Paths, or Both while composing; image and strokes share the same transform. A layer may overflow the page in the preview so placement stays easy, while SVG export and plotting clip that overflow cleanly to the paper.

Crop & masks

Two ways to show only part of a layer, both non-destructive:

  • Crop — a rectangle. To content shrinks it to the drawn geometry; Reset clears it.
  • Mask — a free shape: Rectangle, Oval, or the Pen tool for a bézier outline (click to place anchors, drag for curves). Edit re-shapes an existing mask; the handles move and resize it on canvas.

Masks are real geometric clips: exported SVG and the plot contain only the visible strokes, cut exactly at the boundary — no hidden ink.

Occlusion — layers that hide what's behind them

Turn on Occlude below and a layer erases the strokes of every layer beneath it inside its silhouette — a portrait region cleanly knocked out of the background hatching, no manual masking of the layers below. Region-based layers use their AI-selected outline; whole-image layers use their footprint. The clipping is baked into export and plot.

Cavalry live bridge

＋ Cavalry arms a live layer that receives frames from Cavalry (the motion-design app) via the bundled bridge script. In Cavalry, the script panel's button creates a matching A3 composition; every change pushes the current frame straight onto your PlotterForge page — design in Cavalry, compose and plot here. The layer shows a LIVE badge while connected, and reconnecting after a restart offers to continue the same layer or start a new one.

Importing SVG

Drop an .svg on the window (or File → Import image… and pick an SVG): it arrives as a composition layer, normalized to the page in real millimetres. Pixel-based exports (like Cavalry's) are fitted to the page automatically so a 1080 px file doesn't try to plot a metre wide.

Pen preview

View → Pen width & nib draws every stroke at its true pen width — including the calligraphic broad-nib simulation for flat pens (set nib shape and angle in the Pens panel). Turn it off to inspect raw centrelines.
