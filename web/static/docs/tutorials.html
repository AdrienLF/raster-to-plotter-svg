Three drawings, start to finish

These are controlled experiments, not tours. Start with the supplied source, enter the listed values, and stop at each checkpoint. Once your result matches, replace one choice at a time with your own.

Download the tutorial source: tutorial-source.png is a small radial tone study used by the automated tests. Its simple light-to-dark structure makes parameter changes easy to see and keeps every result reproducible.

Photo → Voronoi Stippling → SVG

Outcome: one clean black stipple layer on A3, ready for SVG export. Allow about five minutes for the first run; the exact generation time depends on the active CPU/GPU backend.

  1. Create a clean project. Choose Project → New project… and name it Tutorial · Stipple. A separate project makes every checkpoint reversible and keeps your own work untouched.
  2. Import the source. Use File → Import image… and choose tutorial-source.png. The image becomes a selected raster layer, centred and fitted without cropping.
  3. Open Path Finding. Click the layer’s ⚙, choose Voronoi Stippling, leave Region on Whole image, and choose Paths for Display.

  1. Render the final geometry. Click Apply / Regenerate. Wait until the layer badge says clean and the status bar returns to Ready.
  2. Check the page. Close Path Finding. In Layers choose Area → Fit, set Rotate to 0°, and confirm the selected bounds stay inside the paper.
  3. Set the physical pen. In Pens, keep one enabled black round pen at 0.5 mm. The preview should remain legible at Fit Page zoom.
  4. Export SVG. Choose File → Export SVG. Open the result in a vector viewer and confirm its dimensions match the Drawing Area in millimetres.

Checkpoint

The selected layer reports Voronoi Stippling · clean; the centre contains the densest dots, the white corners are nearly empty, and the status bar shows a non-zero shape count and path length.

If your result differs: confirm the source is the supplied PNG, Region is Whole image, Ignore White is on, Seed is 0, and you clicked Apply / Regenerate rather than judging the low-resolution live draft.

Generator → multi-pen poster

Outcome: an A3 Spokes & Circles composition distributed across three physical pens. This route needs no image.

  1. Create Tutorial · Generator, click ＋ Generator, and select Spokes & Circles.
  2. In Drawing Area choose A3 portrait. In Pens add three enabled pens named Dark, Mid, and Accent, all at their real measured widths.
  3. Enter the values below, then click ✦ Generate. Generator switching is deliberate: changing the generator itself never redraws until you press Generate.

Checkpoint

The page contains eight radial clusters and background rays; the Pens panel shows non-zero usage for all three enabled pens. Save a Version named Baseline · 3 pens before experimenting with rotation or distortion.

If your result differs: regenerate after editing the enabled pen list. Pen cycling follows the current Drawing Set; a disabled pen cannot receive geometry.

Estimate and Pre-flight

  1. Open 3 Plot and inspect Estimate. A shape count, drawn distance, travel distance and duration must all be present.
  2. If travel dwarfs drawing distance, try Nearest + reverse paths. Re-estimate before choosing 2-opt on dense work.
  3. Confirm paper size and orientation, then Home, jog to the intended origin, and test Pen up/down over scrap.
  4. Use ▶ Preview to watch pen order and travel moves. Only then load the final paper and press Start.
